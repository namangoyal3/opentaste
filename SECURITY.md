# 🔒 Security Policy

## Reporting a Vulnerability

We take security seriously. If you find a vulnerability, please email namangoyal21197@gmail.com.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours.

## Scope

- The OpenTaste CLI and packages
- The web dashboard

## Out of Scope

- Third-party dependencies (report to their maintainers)
- The AI tools that consume generated context files
